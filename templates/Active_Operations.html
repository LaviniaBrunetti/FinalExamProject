Operations

-Get Columns: Returns the name of the columns present in the Dataset as well as the Data Type for each column.

-Get Unique ID: Returns all the seqID present in the Dataset.

-Get Unique Type: Returns all the values for the column "Type" present in the Dataset.

-Get Number of Entries for Type: Retruns the number of entries for each "Type" present in the Dataset

-Get Whole Chromosome Dataset: Returns a new Dataset containing only the entries of full chromosomes

-Get Fraction of Fragmented Features: Returns the number of entries of entire chromosomes, the number of entries of fragmented (ID:supercontig) and the number of other features, as well as a list of unique seqIDs for these features and the fraction of fragmented/tot.

-Get Ensembl-Havana Dataset: Returns a new Dataset containing only the entries with Source Ensembl, Havana or Ensembl-Havana. Please notice that only 2000 entries are shown when the result is visualized, as visualizing over 2 million rows would take eccessive runtime.

Count Entries in Ensembl-Havana Dataset: Returns the number of entries for each Source.

-Get Genes Names: Returns the Name of each gene in the DataFrame as well as the index of such entries in the DataSet
